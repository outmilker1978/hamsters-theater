## Для кого эта программа

TV Hamsters — это программа для **совместного просмотра видео онлайн**. Один включает трансляцию своего экрана со звуком, остальные видят и слышат всё в реальном времени. При этом одновременно Все участники могут общаться голосом через микрофон, включать свои камеры и переписываться.

Подходит для:

- **Киновечера с друзьями** — включили фильм, смотрите и комментируете вместе
- **Учёбы онлайн** — преподаватель показывает экран, ученик повторяет
- **Общения с теми, кто далеко** — день рождения, Новый год вместе в разных городах
- **Игровых стримов для своих** — показываешь игру, друзья комментируют

Характеристики:

| Параметр | Значение |
|---|---|
| Макс. участников | 5 |
| Соединение | P2P (напрямую между участниками) |
| Сигнальный сервер | облачный (Render.com) + встроенный локальный |
| Десктоп | Windows, портативный .exe |
| Мобильная версия | PWA (работает в браузере, можно установить) |

## Как это работает

```
Участник A (Desktop) ───── P2P ──── Участник B (Desktop)
         │                                    │
         ├───── P2P ───────────────────────────┤
         │                                    │
    Участник C (Mobile) ──── P2P ─────────────┘
                │
           Сервер сигналинга (только для установки соединения)
```

1. **Сервер сигналинга** помогает участникам найти друг друга (облачный сервер или локальный)
2. После нахождения **данные (видео, аудио, экран) передаются напрямую** между участниками — без задержек и без нагрузки на сервер
3. Достаточно, чтобы **канал интернета** у каждого участника был стабильным (хотя бы 2-5 Мбит/с)
4. **STUN/TURN** сервера помогают пробить NAT, если участники за разными роутерами

1. Скачай `TV.Hamsters.X.Y.Z.exe` со страницы релизов

Рядом с иконкой каждого участника — ползунок громкости (всегда виден). Можно настроить громкость каждого хомячка отдельно.

|---|---|
| **Файл** → Выход | Закрыть программу |
| **Настройки** → Режимы | Переключение Локально / Через Интернет |
| **Настройки** → Полный экран | Полноэкранный режим окна |
| **Настройки** → Инструменты разработчика | Консоль для отладки |

### Сочетания клавиш

| Клавиша | Действие |
|---|---|
| **Enter** (в поле кода) | Войти в комнату |
| **Пробел** (в режиме рации) | Говорить (зажать) |
| **Средний клик мыши** | Полноэкранный режим |

### Плавающая панель (при трансляции экрана)

Когда кто-то транслирует экран, у шарера появляется **маленькое окно-панель** (поверх всех окон) с кнопками управления: микрофон, камера, рация, экран, выход. Можно управлять, не переключаясь в главное окно.

### Окно камер участников (при трансляции)

Рядом с панелью открывается **окно камер** (300×220, внизу справа). Показывает превью камер всех участников с ползунками громкости. Обновляется 5 раз в секунду.

## Mobile-версия (PWA)

### Как открыть

**Важно:** TV Hamsters использует WebRTC — технологию видеосвязи, встроенную в браузер.
Она работает на движке Chromium (Google Chrome, Яндекс.Браузер, Samsung Internet, Opera и другие).
**Рекомендуем Google Chrome** — он обеспечивает наилучшую совместимость и стабильность P2P-соединения.
Другие браузеры (особенно Яндекс.Браузер) могут иметь ограничения: встроенный VPN/Турбо-режим, блокировка WebRTC или нестандартное поведение разрешений камеры/микрофона.

**Вариант 1 — в браузере:**
Открой `https://tvhamsters.outmilk.online/mobile/` в Chrome или Safari.

**Вариант 2 — установить на телефон (Android):**
1. Открой сайт в Chrome
2. Нажми **«Скачать на Android»** (кнопка появляется автоматически)
3. Или: меню Chrome → «Добавить на главный экран»
4. После установки приложение работает без адресной строки

## Сценарии использования

### 1. Кинотеатр на диване

**Кто шарит (Desktop):**
1. Создай комнату
2. Друзья вводят код
3. Нажми «Экран» → выбери окно с плеером (фильм, сериал, YouTube)
4. Говори в микрофон, комментируй

**Кто смотрит (Desktop/Mobile):**
1. Ввели код → подключились
2. Видят трансляцию экрана и слышат голос
3. Могут включить свою камеру/микрофон, чтобы обсуждать

### 2. Наставник онлайн (удалёнка)

1. Преподаватель транслирует экран с программой
2. Ученик видит экран и слышит голос
3. **Режим рации** — ученик зажимает Пробел, чтобы задать вопрос, и отпускает, когда слушает
4. Идеально для: обучение софту, дизайн, программирование, испанский язык

### 3. Ночные посиделки с друзьями

1. Разошлись по городам, а хочется вместе
2. Один включает трансляцию экрана — можно вместе смотреть видео, мемы, стримы
3. Микрофон включён постоянно — можно переговариваться
4. До 5 человек в одной «палате»

### 4. Общение без видео (только голос)

- Если не хочешь включать камеру — просто отключи её
- Микрофон можно включать/выключать по необходимости
- Режим рации — для минимального шума в эфире

## Рация (Push-to-Talk)

Режим, в котором микрофон **включён только пока держишь кнопку**. Отпустил — микрофон выключен, тебя не слышно.

### Desktop
1. **Правая кнопка мыши** по иконке микрофона → вход в режим рации
2. **Пробел** (зажать) → говорить
3. Отпустить Пробел → микрофон выключен
4. **Любой клик (левый/правый)** по иконке микрофона → выход из режима

### Mobile
1. **Двойной тап** по кнопке микрофона → вход в режим рации
2. Появится кнопка <svg width="18" height="18" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2"><rect x="2" y="6" width="20" height="12" rx="2"/><path d="M6 6V4a2 2 0 0 1 2-2h8a2 2 0 0 1 2 2v2"/></svg> — **зажми и говори**
3. Отпусти — микрофон выключен
4. **Двойной тап** по кнопке рации или по микрофону → выход из режима

## Трансляция экрана со звуком

### Desktop
1. Нажми **«Экран»**
2. Откроется нативный диалог выбора окна (Chromium)
3. Выбери **окно** или **весь экран** (можно выбрать даже свёрнутое окно!)
4. Tick **«Поделиться звуком»** чтобы передавать звук системы
5. Нажми «Поделиться»
6. Главное окно свернётся — появится **плавающая панель** и **окно камер участников**
7. Чтобы завершить — нажми «Экран» на панели

### Mobile
- Мобильная версия **не транслирует экран** (техническое ограничение браузеров)
- Мобила **может смотреть** трансляцию от Desktop-участников

## Решение проблем

### Не удаётся подключиться

| Проблема | Решение |
|---|---|
| «Не удаётся подключиться к серверу» | Проверь интернет. В России может потребоваться VPN для Render.com |
| «Комната не найдена» | Проверь код комнаты. Комната существует, пока создатель в ней |
| Нет звука | Проверь микрофон/наушники. На Mobile проверь разрешения в браузере |
| Нет видео | Проверь камеру. В браузере дай разрешение на использование камеры |

### Desktop

| Проблема | Решение |
|---|---|
| Не запускается .exe | Скачай новую версию. Windows Defender может блокировать — нажми «Сохранить всё равно» |
| Нет звука системы при трансляции | В диалоге выбора окна включи «Поделиться звуком» |
| Эхо/обратная связь | Используй наушники, а не колонки |
| Ошибка подписи при сборке | Не обращай внимания — портативная версия работает без подписи |

### Mobile

| Проблема | Решение |
|---|---|
| Не вижу и не слышу собеседника | Используй Google Chrome (не Яндекс.Браузер). Отключи Турбо-режим и встроенный VPN в настройках браузера |
| Страница не обновляется | Открой в приватной вкладке или очисти кэш браузера |
| Кнопка «Скачать на Android» не появляется | Используй Chrome, открой меню → «Добавить на главный экран» |
| Полный экран показывает оверлей Chrome | Установи PWA на телефон (не в браузере) |
| Нет звука в PTT | Дай разрешение на микрофон при запросе браузера |

### Соединение

| Проблема | Решение |
|---|---|
| P2P не соединяется (висят «Подключение...») | Используй режим «Через Интернет», а не «Локально» |
| Задержка видео | Проверь скорость интернета. Закрой лишние вкладки |
| Участник отключился | Нажми «Выйти» и зайди заново. Если потерял код — создай новую комнату |
