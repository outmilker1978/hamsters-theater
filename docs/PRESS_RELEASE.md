# Вышла мобильная версия TV Hamsters — теперь смотрите кино вместе с телефона

**TV Hamsters, бесплатное приложение для совместного просмотра видео,**
**объявляет о выходе мобильной версии.**

Мобильная версия TV Hamsters — это полноценный клиент для просмотра и участия
в комнатах. Без установки, без регистрации. Просто открываете ссылку в браузере —
и вы в комнате.

Работает как Progressive Web App (PWA) на Android и iOS. Можно установить
на экран дома и использовать как обычное приложение.

## Что есть в мобильной версии

- **Вход по коду** в уже созданную комнату
- **Создание новой комнаты** прямо с телефона
- **Видеозвонок** — камера и микрофон
- **Режим рации (PTT)** — двойной тап по микрофону
- **Управление звуком** — ползунки громкости каждого участника
- **Полный экран** — просмотр на всю ширину экрана
- **Поделиться** — отправить код комнаты друзьям
- **Установка на Android** — как обычное приложение с иконкой

## Как это работает

Всё то же самое, что и в десктопной версии: P2P-соединение напрямую между
участниками, никаких серверов для видео, всё бесплатно.

Мобильная версия предназначена в первую очередь для просмотра.
Трансляция экрана со звуком — уникальная возможность Windows-версии,
а для мобильных устройств мы сделали удобного "зрителя".
