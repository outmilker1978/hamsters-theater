# Продвижение на софт-порталах — инструкция

## Что это

Сайты-каталоги, где люди ищут бесплатные программы под Windows.
Достаточно один раз добавить приложение — и оно висит там годами,
принося скачивания из поиска и категорий.

---

### Шаг 2. Добавить программу — что вставлять в поля

Вот готовые тексты для всех полей. Копируй и вставляй.

#### Название программы

```
TV Hamsters
```

#### Краткое описание (одна строка, 1-2 предложения)

Russian:
```
Бесплатная программа для совместного просмотра видео онлайн. Трансляция экрана со звуком, голосовой чат, видеозвонок и рация. Portable .exe, без установки.
```

English:
```
Free desktop app for watching videos together online. Share your screen with system audio, voice chat, video call, and push-to-talk. Portable .exe, no installation required.
```

#### Полное описание (вставь это в поле Description)

Russian (основной):
```
TV Hamsters — программа для совместного просмотра видео онлайн.

Главная возможность — трансляция экрана компьютера со звуком системы.
Один включает показ своего экрана (фильм, стрим, презентацию),
остальные участники видят и слышат всё в реальном времени.

Возможности:
• Трансляция экрана со звуком системы
• Голосовой чат через микрофон
• Видеозвонок (камера каждого участника)
• Режим рации Push-to-Talk (зажал пробел — говоришь)
• До 5 участников в одной комнате
• P2P-соединение — видео и голос идут напрямую, без сервера
• Два режима: локально (LAN) или через Интернет
• Работает без установки (портативный .exe)
• Без рекламы, без регистрации, без следилок
• Мобильная версия (PWA) для просмотра с телефона

Как использовать:
— Киновечера с друзьями
— Удалённое обучение (преподаватель показывает экран)
— Игровые стримы для своих
— Общение с теми, кто далеко

Сайт: tvhamsters.outmilk.online
```

English (для международных порталов):
```
TV Hamsters — a free desktop app for watching videos together online.

The main feature is screen sharing with system audio. One person shares
their screen (movie, stream, presentation), others see and hear everything
in real time.

Features:
• Screen sharing with system audio
• Voice chat via microphone
• Video call (each participant's camera)
• Push-to-Talk mode (hold spacebar to talk)
• Up to 5 participants per room
• P2P connection — video and audio go directly between participants
• Two modes: local (LAN) or over the Internet
• Portable .exe, no installation required
• No ads, no registration, no tracking
• Mobile version (PWA) for viewing on phone

Website: tvhamsters.outmilk.online
```
